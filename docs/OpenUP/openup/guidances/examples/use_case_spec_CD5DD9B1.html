Main Description

The attached document provides an example of a use-case specification for an Automated Teller Machine (ATM). The ATM was selected as an example system because it is familiar to most people. This familiarity simplifies understanding the principles without getting lost in domain-specific terminology.

A companion example, Evolution of the Use-Case Model, shows how the use-case model as a whole evolves over time.
