Task: Keep Process On Track
Purpose
  • Ensure the process is being used effectively towards the project goals.
Relationships
RolesPrimary Performer:
  • XP Coach
Additional Performers:
Steps
General

The Whole Team needs to be very focused on the goals of the project. The development practices enable the project goals to be achieved. The challenge of focusing on the project goals and focusing the disciplined and effective use of the practices is a constant challenge for the team.

The XP Tracker collects data and reports it to the team. This data provides guidance as to whether the project is on a trajectory to meet the project goals. The data also monitors the team's use of the practices.

The XP Coach ensures that the team is collecting and using the data to ensure that the project and process goals are achieved.
