Task: Run Customer Test
Purpose
  • Execute the automated acceptance test and gather results.
Relationships
RolesPrimary Performer:
  • XP Tester
Additional Performers:
InputsMandatory:
  • None
Optional:
  • XP Customer Test
    Steps
    General

    This task consists of running all the customer or acceptance tests to ensure that all previously completed user stories are still working. A lot of XP teams run those tests on a daily basis as part of their automated daily build. Other teams run those tests every time a new story is added to the pool of completed stories to ensure that a new story has not caused previous ones to fail.
