Role: XP Programmer (Administrator)
The XP Programmer (Administrator) is responsible for managing the programmer environment.
Role Sets: XP Roles
Relationships
Primary Performs
  • Setup Programmer Environment
Main Description

The XP Programmer (Administrator) role includes most of the traditional software development technical roles, such as designer, implementer, integrator, and administrator.

In the administrator role, the programmer deals with establishing the physical working environment.

Staffing
Skills

The XP Programmer (Administrator) role is responsible for establishing the workspace for pair programming, including removing cubicle walls, establishing line of sight with the customer, and standardizing on the development tools.
