Concept: Change Requests
A change request is a general term for any request to change a work product.

Main Description

A change request represents any request to change a work product. This includes items commonly called defect reports, enhancement requests, requirements change request, implementation requests, and stakeholder requests.

In this process, change request are captured in the Artifact: Work Items List.  See Guideline: Managing Work Items for more information on the recommended attributes of change requests.
